The CSS Book

The CSS Book

by yaxxei

Этот справочник представляет собой удобный инструмент для изучения и быстрого поиска информации о CSS (Cascading Style Sheets). Он содержит структурированное описание ключевых тем, начиная с основ и заканчивая продвинутыми возможностями.

Данный справочник предполагает, что вы используете CSS3.

Справочник подходит для новичков и опытных разработчиков, стремящихся улучшить свои навыки работы с CSS.
